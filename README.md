# Cfrolich13 Tap-iloader

## How do I install these formulae?

`brew install cfrolich13/tap-iloader/<formula>`

Or `brew tap cfrolich13/tap-iloader` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "cfrolich13/tap-iloader"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
